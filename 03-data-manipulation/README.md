# Data Manipulation

The core of any data science workflow: loading, cleaning, transforming, and exploring data using NumPy and Pandas.

## Prerequisites

- Python foundations
- Basic understanding of tabular data

## Learning Objectives

- Manipulate multi-dimensional arrays with NumPy
- Perform vectorized operations for performance
- Clean and transform real-world messy datasets with Pandas
- Handle missing values, outliers, and data type issues
- Merge, join, concatenate, and reshape DataFrames
- Apply group-by operations for aggregation and transformation
