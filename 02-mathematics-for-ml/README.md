# Mathematics for Machine Learning

The mathematical foundations needed to understand how machine learning algorithms work under the hood.

## Prerequisites

- High school algebra
- Basic programming

## Learning Objectives

- Understand vectors, matrices, and their operations
- Grasp eigenvalues, eigenvectors, and matrix decomposition
- Master probability fundamentals (Bayes theorem, distributions)
- Apply statistical concepts (hypothesis testing, correlation, p-values)
- Understand calculus concepts used in gradient descent

## Key Topics

### Linear Algebra
- Vectors: dot product, norm, cosine similarity
- Matrices: multiplication, transpose, inverse, determinant
- Decompositions: Eigendecomposition, SVD, Cholesky
- Applications: PCA derivation, linear regression normal equation

### Probability
- Probability axioms, conditional probability, Bayes theorem
- Random variables (discrete/continuous)
- Common distributions: Normal, Binomial, Poisson, Uniform, Exponential
- Maximum Likelihood Estimation (MLE)

### Statistics
- Descriptive statistics: mean, median, mode, variance, skewness, kurtosis
- Inferential statistics: confidence intervals, hypothesis testing
- Correlation: Pearson, Spearman, mutual information
- The Central Limit Theorem and its importance

### Calculus
- Derivatives, partial derivatives, gradient
- Chain rule for backpropagation
- Optimization: gradient descent, stochastic GD, momentum
