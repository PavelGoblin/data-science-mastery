# House Price Prediction

A complete regression project using the Ames Housing dataset. Covers the full ML workflow from EDA to model deployment.

## Key Techniques
- Feature engineering (combining rare categories, creating interaction features)
- Handling missing values (median imputation, mode imputation)
- Log transformation of skewed target variable
- Regularization (Ridge, Lasso, ElasticNet)
- Ensemble models (Random Forest, Gradient Boosting, Stacking)
- Hyperparameter tuning with RandomizedSearchCV

- `house-price-prediction.ipynb` — full analysis and modeling notebook
