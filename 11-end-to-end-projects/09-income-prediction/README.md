# Income Prediction

Binary classification project: predict whether an individual's annual income exceeds $50K based on demographic and employment features (age, education, occupation, hours-per-week, marital status, race, sex, native-country, capital gains/losses).

Uses synthetic census-like data (via `make_classification`) to simulate the classic UCI Adult Income dataset. Compares multiple classifiers with cross-validation, hyperparameter tuning, and bias analysis.

**Notebook:** `income-prediction.ipynb`
