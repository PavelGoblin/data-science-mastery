# Customer Segmentation

This project applies unsupervised learning techniques to segment customers for targeted marketing. Synthetic customer data is generated with features including age, income, spending score, purchase frequency, recency, loyalty score, and average order value.

**Methods used:**
- K-Means (Elbow + Silhouette analysis)
- Hierarchical clustering (dendrogram)
- DBSCAN (outlier detection)
- Gaussian Mixture Models (soft clustering)

**Key outputs:** Segmented customer profiles, 2D PCA projections, radar charts, and marketing recommendations per segment.
