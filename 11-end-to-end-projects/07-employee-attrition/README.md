# Employee Attrition Prediction

End-to-end ML project to predict which employees are likely to leave the company. Uses synthetic HR data generated with `make_classification` to simulate realistic employee attributes (age, tenure, job satisfaction, salary, department, overtime, work-life balance, etc.). Compares LogisticRegression, RandomForest, GradientBoosting, and DecisionTree models with class imbalance handling, hyperparameter tuning, threshold tuning, and business recommendations.
