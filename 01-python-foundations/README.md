# Python Foundations for Data Science

Master Python fundamentals through a data science lens. This module covers everything from basic syntax to advanced features like decorators, generators, and context managers — all with practical data science examples.

## Prerequisites

- Basic programming knowledge (any language)
- Python 3.9+ installed

## Learning Objectives

- Write idiomatic, efficient Python code
- Understand and use functional programming tools (map, filter, reduce, lambdas)
- Master list/dict/set comprehensions and generator expressions
- Use context managers for resource handling
- Apply decorators for logging, timing, and caching
- Leverage Python's built-in data structures effectively

## Key Topics

1. Data Types & Structures — lists, tuples, dicts, sets, frozensets

3. Functions — args, kwargs, closures, decorators
4. Iteration — iterators, generators, itertools
5. OOP — classes, inheritance, magic methods, dataclasses
6. Error Handling — try/except/else/finally, custom exceptions
7. Modules & Packages — imports, __init__.py, namespace packages
8. Performance — profiling with cProfile, memory profiling, __slots__
