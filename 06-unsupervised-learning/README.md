# Unsupervised Learning

Finding hidden patterns in unlabeled data through clustering, dimensionality reduction, and anomaly detection.

## Prerequisites

- Supervised learning basics
- Linear algebra fundamentals

## Learning Objectives

- Group similar data points with clustering algorithms
- Reduce dimensionality for visualization and compression
- Detect anomalies and outliers in data
- Evaluate clustering quality without ground truth labels
